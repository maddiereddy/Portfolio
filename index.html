Opa Grappa

Opa Grappa is an app that keeps track of all the wines that a user has consumed or would like to try in the future. You can search for a wine by narrowing down to a winery or vineyard and can find more information of a selected wine like the name, type, vintage, vineyard, ratings and reviews (by Wine Enthusiast) and add your own comments!

PlugHubs

Helps users locate electric vehicle charging stations near them (using geolocation), or at a given address or zip code, using the NREL API with Google Maps Javascript API. You can filter the locations by: distance, type of connector, network or charging levels. Its great for planning a road trip and noting ev charging stations along the way!
